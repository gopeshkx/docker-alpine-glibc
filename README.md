## Minimal Alpine Linux Docker image with glibc [![Circle CI](https://circleci.com/gh/jeanblanchard/docker-alpine-glibc/tree/alpine3.10.svg?style=shield)](https://circleci.com/gh/jeanblanchard/docker-alpine-glibc/tree/alpine3.10)

Basic [Docker](https://www.docker.com/) image based on [Alpine Linux](http://alpinelinux.org/), with [glibc](https://github.com/sgerrand/alpine-pkg-glibc).

### Support

The 3.10 branch of Alpine Linux will receive updates until 2021-05-01 .
